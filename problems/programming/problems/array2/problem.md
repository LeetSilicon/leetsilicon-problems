## Problem Statement

Reverse an Array

Given an array `nums`, reverse it **in-place**.\n\n' +
        '**Example 1:**\n' +
        '```\nInput: nums = [1,2,3,4,5]\nOutput: [5,4,3,2,1]\n```\n\n' +
        '**Example 2:**\n' +
        '```\nInput: nums = [1,2,3,4]\nOutput: [4,3,2,1]\n```\n\n' +
        '**Constraints:**\n' +
        '- `0 <= nums.length <= 10⁵`\n' +
        '- O(1) extra space

---

## Requirements

1. Reverse in-place with O(1) extra space

2. Use two-pointer technique
