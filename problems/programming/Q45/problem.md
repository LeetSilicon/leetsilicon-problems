# Count Vowels

**Domain:** programming — Array Problems  
**Difficulty:** Easy  
**Topics:** String

---

## Problem Statement

Count Vowels

Given a string `s`, return the number of vowels (`a, e, i, o, u`, case-insensitive).\n\n' +
        '**Example 1:**\n' +
        '```\nInput: s = "hello"\nOutput: 2\n```\n\n' +
        '**Example 2:**\n' +
        '```\nInput: s = "aeiou"\nOutput: 5\n```\n\n' +
        '**Constraints:**\n' +
        '- `0 <= s.length <= 10⁵`

---

## Requirements

1. Case-insensitive

2. Ignore non-letter characters

---

## Hints

<details>
<summary>Hint 1</summary>
Lowercase the character and check membership in {a,e,i,o,u}.
</details>
