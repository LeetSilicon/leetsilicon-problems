## Problem Statement

To Lower Case / Upper Case

        '**Example 1:**\n' +
        '```\nInput: s = "Hello"\nOutput:\nhello\nHELLO\n```\n\n' +
        '**Constraints:**\n' +
        '- `1 <= s.length <= 100`\n' +
        '- Non-alphabetic characters stay unchanged

## Requirements

1. Implement both conversions
