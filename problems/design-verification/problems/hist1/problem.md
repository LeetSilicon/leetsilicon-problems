# Last 4 Values Unique

**Domain:** design-verification — Constraints  
**Difficulty:** Medium  
**Topics:** Constraints, Stream

---

## Problem Statement

Stream with Last 4 Generated Values All Different

Write uvm sv constraints to generate stream of random numbers where the most recent four values are always unique (no repeats in last 4). Maintain sliding window history. Define value domain large enough for feasibility (≥5 distinct values).

---

## Requirements

1. STREAM: Sequence of random values generated one at a time.

2. SLIDING WINDOW: Maintain history of last 4 generated values.
